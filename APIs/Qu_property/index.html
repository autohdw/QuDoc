    Qu的成员

    Info

    所有的Qu的成员全部为公有，可以直接访问。

    data

    int

    以32位整形实际存储的数据。

    Warning

    • 请不要直接折腾data，除非你知道自己在干什么。
    • 无论是否要求了符号位，data都是有符号的。

    toDouble()

    double

    返回实际代表的double值。

    Example

    Qu<intBits<2>, fracBits<2>, isSigned<true>> a = 0.92;
std::cout << a.toDouble() << std::endl;
    输出： 
    0.75

    toString()

    std::string

    返回比特串的字符串表示。

    fill(·dis·int)

    Qu

    将Qu填充为指定的值。将会返回*this，以便链式调用。

    无参数时，将会填充可能范围内的随机均匀分布。

    Example

    Qu<> a;
a.fill();

    可以传入一个分布，将会填充为分布的值。

    QuBLAS.h中预设了UniRand和NormRand两个分布。

    Example

    Qu<> a;
a.fill(NormRand);

Qu<> b;
b.fill(UniRand);

Qu<> c;
std::normal_distribution<double> myDis(1, 2);
c.fill(myDis);

    传入一个整数，将会填充为这个整数。

    这里推荐使用二进制字面量，可以更直观的看到填充的值。

    Example

    Qu<> a;
a.fill(0b1010);
a.display();
    输出： 
    intBits: 8 fracBits: 8 isSigned: 1 Static
Binary: 00000000000001010
Binary 32: 00000000000000000000000000001010
Hex: a
Decimal: 0.0390625

    display()

    打印debug信息。

    Example

    Qu<intBits<8>, isSigned<true>> a = 1;
a.display();
    输出： 
    intBits: 8 fracBits: 12 isSigned: 1 Static
Binary: 000000001000000000000
Binary 32: 00000000000000000001000000000000
Hex: 1000
Decimal: 1

    operator <<

    ofstream

    重载<<运算符，用于输出到流。将会输出实际代表的toDouble()值。

    Example

    Qu<intBits<2>, fracBits<2>, isSigned<true>> a = 0.92;
std::cout << a << std::endl;
    输出： 
    0.75

    operator = Any

    Qu

    重载=运算符，用于赋值。

    这里的Any可以是任何数值类型。

    可以从Qu赋值给Qu，也可以从任意实数类型赋值给Qu，比如int、double等。

    需要注意的是，来自同样配置的Qu时会直接拷贝data，而来自不同配置的Qu时会进行转换，此时将会进行量化和溢出判断。

    Example

    Qu<intBits<4>, fracBits<4>, isSigned<true>> a;
a = 0.92;

Qu<intBits<3>, fracBits<3>, isSigned<true>> b;
b = a; // 会进行量化和溢出判断

    大部分成员和实数标量一样。

    real

    Qu

    实部。

    imag

    Qu

    虚部。

    fill(disint)

    Qu

    将复数填充为指定的值。将会对实部和虚部分别调用fill()。

    size

    dim<N...>

    张量的大小。需要注意的是，这是一个dim<N...>类型，不是一个值。

    data

    std::array<Qu, N>

    以std::array存储的数据。其中N为张量的大小。可以通过size::elemSize获取元素个数。

    operator [size_t ...]

    Qu

    重载[]运算符，用于访问张量的元素。

    具有两种重载形式：索引数量等于维度，或者索引仅为一个。

    索引数量等于维度时，就是取出对应坐标的元素。

    Example

    Qu<dim<2,2>, Qu<>> mat = {1, 2, 3, 4};
std::cout << mat[0, 1] << std::endl;

    输出： 

    2

    索引数量为单个索引时，是直接对data进行索引。

    这等效与data[index]。也就是Matlab中的单索引操作。

    Example

    Qu<dim<2,2>, Qu<>> mat = {1, 2, 3, 4};
std::cout << mat[2] << std::endl;
    输出： 
    3

    display()

    打印debug信息。效果为对每个元素调用display()。

    clear()

    将张量清零。

    fill(disint)

    Qu

    将张量填充为指定的值。

    和实数标量的fill()一样，只是这里是对每个元素都调用一次fill()。

    shuffle()

    Qu

    将张量打乱。将会返回*this，以便链式调用。

    toDouble()

    std::array<double, N>

    返回一个std::array<double, N>，其中N为张量的大小。每个元素都是对应元素的toDouble()值。

    operator = Any

    Qu

    重载=运算符，用于从任何来源赋值。

    要求来源是一个可以被方括号索引的类型，比如std::array、std::vector或者另一个Qu张量。

    运行的过程是对范围为0 : size::elemSize的每个i调用来源的[i]索引，然后赋值给对应的元素。

    Warning

    请自行确保来源的大小和张量的大小一致，不会进行检查。

    Example

    Qu<dim<2,2>, Qu<>> mat;
std::array<double, 4> arr = {1, 2, 3, 4};
mat = arr;

std::vector<double> vec = {1, 2, 3, 4};
mat = vec;

mat = {1, 2, 3, 4};

    operator <<

    ofstream

    重载<<运算符，用于输出到流。不同于toDouble()，这里的打印会尽可能地保持一个工整的格式。

    Example

    Qu<dim<2,2>, Qu<>> mat = {1, 2, 3, 4};
std::cout << mat << std::endl;
    输出： 
    [1.0000, 3.0000
 2.0000, 4.0000]
